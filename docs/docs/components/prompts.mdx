# Prompts


### PromptTemplate


Schema to represent a prompt for an LLM.

A `PromptTemplate` refers to a reproducible way to generate a prompt. It contains a text string called "the template" that can take in a set of parameters from the end user and generates a prompt. A prompt template can include instructions to the language model, a set of few-shot examples to help the language model generate a better response, and a question to the language model. It provides a flexible way to create prompts for language models.


- **template:** Template used to format an individual request.

### ZeroShotPrompt


Prompt template for Zero Shot Agent.

Similar to the `PromptTemplate`, the `ZeroShotPrompt` is a template specific to interact with Zero Shot Agents. The template can contain instructions to the language model, few-shot examples to help the model generate a better response, and a question to the model. It provides a flexible way to create prompts for the Zero Shot Agent.

- **format_instructions:** The format of the prompt template. It can have custom format, which will pass instruction to the language model.

Defaults to:

```
Use the following format:

Question: the input question the user must answer

Thought: the user should always think about what to do

Action: the action to take, should be one of [{tool_names}]

Action Input: the input to the action

Observation: the result of the action

... (this Thought/Action/Action Input/Observation can repeat N times)

Thought: I now know the final answer

Final Answer: the final answer to the original input question
```


- **prefix:** A prompt template string to put before the examples.

Defaults to:

```
Answer the following questions as best the user can. The user has access to the following tools:
```


- **suffix:** A prompt template string to put after the examples.

Defaults to:

```
Begin!

Question: {input}

Thought:{agent_scratchpad}
```
