# Utilities

Utilities are a set of actions that can be used to perform common tasks in a flow. They are available in the **Utilities** section in the sidebar.

---

### Bing Search API Wrapper

Wrapper for Bing Search API.

**Params**

- **Bing Search URL:** The URL for the Bing Search API. (Type: str)
  
- **Bing Subscription Key:** The subscription key for accessing the Bing Search API. (Type: str)

- **Number of results:** Number of results to retrieve. (Type: int, Default: 10)


---

### Google Search API Wrapper

Wrapper for Google Search API.

**Params**

- **Google API Key:** The API key for accessing the Google Search API. (Type: str)
  
- **Google CSE ID:** The Custom Search Engine (CSE) ID for using Google Custom Search. (Type: str)

---

### Google Serper API Wrapper

Wrapper around the Serper.dev Google Search API.

**Params**

- **Result Key for Type:** Specifies the result key for different types of search results. (Type: dict)

- **Serper API Key:** The API key required for accessing the Serper.dev Google Search API. (Type: str)

---

### JSON Document Builder

Builds a Document containing a JSON object using a key and another Document's page content.

**Params**

- **Key:** The key to use for the JSON object.
- **Document:** The Document from which to extract page content.

Utilize this component to efficiently structure data within documents by converting page content into JSON format.

---

### Search Api

Interacts with the SearchApi real-time search engine results API.

**Params**

- **Engine:** The search engine to use.
- **Parameters:** The parameters to send with the request.
- **API Key:** The API key to use with SearchApi.

Use this component to integrate real-time search engine results from SearchApi into your workflow with ease.

    <p>
        Detailed information about this API can be found in the [documentation](https://www.searchapi.io/docs/google)
    </p>
</Admonition>

### Searx Search Wrapper

Wraps the Searx API for performing searches.

**Params**

- **Headers:** Custom headers to include in the request. Default: `{"Authorization": "Bearer <token>"}`.
- **k:** The number of results to retrieve. Default: 10.
- **Searx Host:** The URL of the Searx instance to use. Default: "https://searx.example.com".

---

### Serp API Wrapper

Wraps the SerpAPI for accessing search engine results.

**Params**

- **SerpAPI API Key:** Your SerpAPI API key for authentication.
- **Parameters:** Additional parameters to customize the search query. Default parameters include engine, google_domain, gl, and hl.

---

### SQL Database

Enables interaction with a SQL database.

**Params**

- **URI:** The URI to the SQL database.

---

### Wikipedia API Wrapper

Wrapper around the WikipediaAPI.

**Params**

- **Top K Results:** The maximum number of results to retrieve. (Default: 3)
- **Language:** The language of the Wikipedia content. (Default: "en")
- **Load All Available Meta:** Whether to load all available meta-information. (Default: False)
- **Max Document Content Characters:** The maximum number of characters for document content. (Default: 4000)

---

### Wolfram Alpha API Wrapper

Wrapper for Wolfram Alpha.

**Params**

- **App ID:** The API key for accessing Wolfram Alpha.
