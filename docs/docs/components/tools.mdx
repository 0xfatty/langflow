### BigSearchRun

A wrapper around Bing Search. Useful for when you need to answer.questions about current events. Input should be a search query. 

Bing Search is a web search engine owned and operated by Microsoft. It provides search results for various types of content, including web pages, images, videos, and news articles. Bing Search uses a combination of algorithms and human editors to deliver relevant and accurate search results to users.


**Params**

- **Api Wrapper**: Wrapper for Bing Search API.

### InfoSQLDatabaseTool

Tool for getting metadata about a SQL database. Input this tool is a comma-separated list of tables, output is the schema and sample rows for those tables. Example Input: `“table1`, `table2`, `table3”`.

**Params**

- **Db:** SQLDatabase to query.

### JsonGetValueTool

Can be used to see value in string format at a given path. Before calling this you should be SURE that the path to this exists. The input is a text representation of the path to the dict in Python syntax (e.g. data["key1"][0]["key2"]).

**Params**

- **Spec:** Getting a value in a JSON spec.


### JsonListKeysTool

Tool for listing keys in a JSON spec. Input is an empty string, output is a comma separated list of tables in the database.

**Params**

- **Spec:** Getting a value in a JSON spec.


### JsonSpec

The `JsonSpec` class is the base class for the JSON spec. It is used to interact with large JSON/dict objects and answer questions about them.

**Params**

- **path:** The parameter refers to the key or keys that are used to access a specific value within a JSON/dict object. It is used to navigate through the structure of the JSON object and locate the desired information. 

- **max value length:** This parameter refers to the maximum allowed length of a value in a JSON document. It is a parameter that can be set to limit the size of values in order to prevent excessively large values from being stored or transmitted.

### News API

Use this when you want to get information about the top headlines of current news stories. The input should be a question in natural language that this API can answer.

**Params**

- **LLM:** Language Model to use in the `News API`.

- **news_api_key:** It refers to the API key that is required to access and authenticate requests to the `News API`. The news_api_key acts as a security measure to ensure that only authorized users can access the `News API` and retrieve news data. [Get your API key here](https://newsapi.org/).

### Podcast API

- **Db:** SQLDatabase to query.


### RequestsDeleteTool

A portal to the internet. Use this when you need to make a DELETE request to a URL. Input should be a specific url, and the output will be the text response of the DELETE request.

**Params**

- **Requests Wrapper:** Lightweight wrapper around requests library `(TextRequestsWrapper)`.


### RequestsGetTool

A portal to the internet. Use this when you need to get specific content from a website. Input should be a url `(i.e. https://www.google.com)`. The output will be the text response of the GET request.

**Params**

- **Requests Wrapper:** Lightweight wrapper around requests library `(TextRequestsWrapper)`.


### RequestsPatchTool

Use this when you want to PATCH to a website. Input should be a json string with two keys: "url" and "data". The value of "url" should be a string, and the value of "data" should be a dictionary of key-value pairs you want to PATCH to the url. Be careful to always use double quotes for strings in the json string The output will be the text response of the 
PATCH request.

**Params**

- **Requests Wrapper:** Lightweight wrapper around requests library `(TextRequestsWrapper)`.

### RequestsPostTool

Be careful to always use double quotes for strings in the json string The output will be the text response of the

### RequestsPutTool

### TMDB API


Useful for when you want to get information from The Movie Database. The input should be a question in natural language that this API can answer.

**Params**

- **LLM:** Language Model to use in the `TMDB API`.

- **tmdb_bearer_token:** It is an authentication token used in the TMDB API. It is used to authorize API requests to the TMDB service. By setting the tmdb_bearer_token in the API request headers, users can make requests to the TMDB API and retrieve information about movies, TV shows, actors, and more. [Get your API key here](https://developers.themoviedb.org/3/getting-started/introduction).


### Tool

Converts a chain, agent or function into a tool. It allows you to define custom tools that can be used by the agent to perform specific tasks.

**Params**

- **description:**  A description of the tool and its purpose.

- **name:** The name of the tool.

- **return_direct:** The parameter refers to the return_direct of the `Tool`. Defaults to `False`.
