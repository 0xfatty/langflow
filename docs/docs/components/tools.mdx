# Tools

Tools are interfaces that an agent, chain, or LLM can use to interact.

### Retriever Tool

This tool provides functionality to interact with a retriever, allowing retrieval of information based on specified criteria and configurations.

**Parameters**

- **Retriever:** The retriever to interact with.
- **Name:** Name of the tool.
- **Description:** Description of the tool.

---

### Search Api

Real-time search engine results API.

**Output Types:** Document

**Parameters**

- **Engine:** The search engine to use.
- **Parameters:** The parameters to send with the request.
- **API Key:** The API key to use SearchApi.
