# Deploy on Railway

Railway is a deployment platform where you can provision infrastructure, develop with that infrastructure locally, and then deploy to the cloud.

Click the following prebuilt [template](https://railway.app/template/Emy2sU?referralCode=MnPSdg).

Click **Deploy Now**.

:::note
Log in using your **GitHub** account to deploy and authorize Railway's access to it. Afterward, accept their Privacy and Data Policy.
:::

Change to your preferred repository name and click **Deploy**.

After clicking on **Deploy**, you should be able to see a deployed URL.

All deployments will appear in the deployments view on your selected service. Clicking on the deployment will bring up the build and deploy logs. Each deployment has the option to configure a Railway-provided domain as well as attach a custom domain.

:::info
Check out the [Railway Docs](https://docs.railway.app/deploy/deployments) for more deployment information.
:::
